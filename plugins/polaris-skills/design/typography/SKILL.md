# Typography

Typography is the art of arranging type in ways that provide innate hierarchy to UI.

## Typography Defines Hierarchy

Typography plays a crucial role in defining hierarchy within design. Variable weights convey different levels of importance.

**Hierarchy techniques:**
- Bolder weights indicate greater significance
- Good type positioning establishes visual prominence
- Size variations create emphasis
- Color differences support hierarchy
- Spacing reinforces relationships

**Do:**
- Use combination of weight, size, color, and positioning
- Create clear visual hierarchy
- Use weights to signal importance
- Support hierarchy through positioning

**Don't:**
- Rely only on color to define hierarchy
- Use only size for emphasis
- Create ambiguous hierarchy
- Miss opportunities for weight variation

## Typography Defines Purpose

Polaris assigns meaning to type based on usage.

**Type assignments:**
- **Mono**: Code and technical content
- **Tabular numbers**: Numbers and currency amounts (fixed-width for alignment)
- **Type scales**: Designed with UI design in mind
- **Specific weights**: Signal specific roles and hierarchy

**Semantic meaning:**
- Type communicates content type
- Consistent styling creates patterns
- Merchants learn type meanings

**Do:**
- Consistently style similar or repeating type
- Use mono for code
- Use tabular numbers for currency and figures
- Assign semantic meaning to type choices

**Don't:**
- Repurpose or reinterpret known type patterns
- Mix type meanings inconsistently
- Change interaction patterns based on type
- Confuse merchants through type inconsistency

## Type Scale System

Shopify uses a carefully designed type scale.

**Principles:**
- Designed for UI efficiency and readability
- Support visual hierarchy
- Maintain consistency
- Optimize for screen reading

## Best Practices

- Use weight variation for hierarchy (bold, regular, light)
- Size should reflect importance
- Color supports but doesn't replace hierarchy
- Positioning and spacing matter
- Consistency across similar elements
- Ensure sufficient contrast for accessibility
- Test readability across devices
