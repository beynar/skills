# Icons

Icons enhance experience by providing intuitive and efficient navigation, conveying information concisely, and making interfaces more visually appealing.

## Clear

Icons are simple and effectively convey their intended meaning.

**Principles:**
- Avoid unnecessary complexity
- Goal is to communicate quickly and clearly

**Do:**
- Draw simple objects that are easy to identify
- Communicate intent immediately

**Don't:**
- Embellish icons with unnecessary details
- Over-complicate icon design

## Consistent

Icons follow a cohesive visual style across the user interface.

**Consistency includes:**
- Consistent line weights
- Consistent shapes and dimensions
- Consistent perspectives
- Consistent general style

**Benefits:**
- Users quickly recognize functionality
- Intuitive navigation
- Cohesive visual language

**Do:**
- Reuse parts of other icons to maintain visual harmony
- Use universally recognized icons
- Maintain style consistency across the set

**Don't:**
- Reinterpret or stylize universally recognized icons
- Break visual harmony with inconsistent styles

## Universal

Universally recognized symbols and metaphors ensure icons are easily understood by broad user base.

**Do:**
- Leverage universally recognized symbols and metaphors
- Create new icons from established references
- Ensure broad user understanding

**Don't:**
- Rely on cultural-specific metaphors
- Use niche or outdated metaphors
- Create confusing symbol associations

## Best Practices

- Keep icon sets visually unified
- Test for clarity across sizes
- Ensure metaphors work cross-culturally
- Prioritize recognition over originality
