# Motion

Motion brings dynamism to an interface, offers visual feedback, and aids merchants' understanding of action outcomes.

## Purposeful

Motion has a clear purpose. It helps merchants understand the interface and results of their actions, not decoration or entertainment.

**Purpose-driven motion:**
- Enhance merchant understanding
- Provide context and clarity
- Signal state changes
- Confirm user actions
- Guide attention

**Do:**
- Use motion to enhance merchant understanding
- Provide context through animation
- Make interactions feel responsive

**Don't:**
- Allow motion to complicate the interface
- Hinder merchant comprehension
- Add decorative motion without purpose
- Distract from core tasks

## Responsive

Motion should be a reaction to merchant interactions, providing immediate visual feedback and making the interface feel alive.

**Responsive motion principles:**
- React to merchant interactions instantly
- Provide visual feedback
- Make interface feel alive and responsive
- Match motion scale to action scale

**Do:**
- Use motion to provide feedback on merchant actions
- Scale motion appropriately to action magnitude
- Provide clear response to interactions
- Make interactions feel connected

**Don't:**
- Use motion that doesn't correspond to interactions
- Provide feedback disconnected from user expectations
- Add motion that confuses the cause-effect relationship

## Snappy

Motion should feel quick and subtle.

**Snappy animation characteristics:**
- Starts rapidly
- Slows down towards the end
- Feels quick while still readable
- Provides cues about element origin
- Feels natural and fluid

**Do:**
- Make motion feel snappy and lively
- Use easing that accelerates then decelerates
- Keep animations energetic
- Maintain quick feedback

**Don't:**
- Use abrupt or instant motion
- Create distracting animations
- Make motion hard to follow
- Slow down feedback too much

## Timing Guidelines

- Short animations: 150-250ms
- Longer animations: 300-500ms
- Keep feedback near-instantaneous
- Use easing curves for natural feel
- Test speed on real devices

## Best Practices

- Motion enhances understanding, not decoration
- Every animation should have clear purpose
- Feedback should be immediate and snappy
- Match motion intensity to action importance
- Test animations with real user interactions
