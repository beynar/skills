# Color Tokens Reference

## Background Colors

### Surface (Primary background elements)
- `--p-color-bg`: rgba(241, 241, 241, 1) - Default background color of the admin
- `--p-color-bg-surface`: rgba(255, 255, 255, 1) - Background for elements with highest prominence (cards)
- `--p-color-bg-surface-hover`: rgba(247, 247, 247, 1) - Hover state for surface elements
- `--p-color-bg-surface-active`: rgba(243, 243, 243, 1) - Active/pressed state for surface elements
- `--p-color-bg-surface-selected`: rgba(241, 241, 241, 1) - Selected state for surface elements
- `--p-color-bg-surface-disabled`: rgba(0, 0, 0, 0.05) - Disabled state color

### Inverse (High contrast backgrounds)
- `--p-color-bg-inverse`: rgba(26, 26, 26, 1) - High contrast page/component backgrounds
- `--p-color-bg-surface-inverse`: rgba(48, 48, 48, 1) - Surface elements on inverse backgrounds

### Transparent
- `--p-color-bg-surface-transparent`: rgba(0, 0, 0, 0) - Fully transparent background
